# 🍁 Mapleki — 매왕 길드 관리 대시보드

## 📌 프로젝트 개요
메이플스토리 키우기 '매왕' 길드(최대 30명)의 길드 컨텐츠 참여도를 관리하기 위한 웹 대시보드입니다.  
길드원의 실제 스펙(전투력 순위)과 컨텐츠 내 수행 등수를 비교하여 '양호/주의/위험' 3단계로 자동 판정합니다.  
기존 엑셀 수작업을 자동화하고, 모바일 및 데스크탑 모두에서 최상의 사용자 경험(UX)을 제공하는 최신화된 인터페이스를 갖추고 있습니다.

## 🎯 주요 기능

### 1. 인증 및 권한
- 환경변수에 저장된 단일 공용 계정(ID/PW)으로 로그인
- 로그인 성공 시 HTTP-Only 쿠키 발급
- **조회(Read):** 로그인 없이 누구나 열람 가능
- **추가/수정/삭제(CUD):** 로그인한 관리자만 접근 가능

### 2. 컨텐츠 및 기록 관리
- **동적 카테고리:** '전체 랭킹' 기준인 서버 컨텐츠인지, 길드 내부 경쟁인지 구분하여 생성
- **기능형 테이블:** 
  - 서버 컨텐츠: `[ 전체 랭킹 | 캐릭터명 | 매왕 등수 | 컨텐츠 등수 | 등수 차이 | 판정 ]`
  - 일반 컨텐츠: 불필요한 데이터 축소 `[ 길드 순위 | 캐릭터명 | 매왕 등수 | 등수 차이 | 판정 ]`

### 3. 극대화된 사용자 편의성 (UI/UX)
- **모바일 완벽 대응:** 햄버거 메뉴(Drawer) 사이드바, `createPortal`을 활용한 안전한 모달 레이아웃, 모바일 화면에서의 끊김 없는 표 스크롤링 등을 지원합니다.
- **Hover-Expand 버튼:** 모든 액션 버튼(작성, 수정, 삭제, 공유 등)은 심플한 아이콘 형태로 존재하며, 마우스를 올릴 때 텍스트가 확장되는 애니메이션을 제공합니다.
- **엑셀형 키보드 내비게이션:** PC에서 등수 입력 시, 마우스 클릭 없이 방향키(↑, ↓)와 Enter 키만으로 상하 셀을 이동하며 빠르게 값을 입력할 수 있습니다. 이미 관리되는 이름과 등수는 텍스트로 보존되어 커서의 불필요한 이동을 막아줍니다.

### 4. 편하게 순위 배정하기 (모달)
- 버튼 하나로 드래그 앤 드롭 모달을 열어 30명의 순위를 직관적으로 조작합니다.
- 좌측의 그립(핸들) 영역만을 드래그 트리거로 설정하여, 모바일 환경에서 화면을 위아래로 편하게 스크롤할 수 있습니다.
- X 버튼 클릭 시 즉시 미참여 탭으로 분류됩니다.

### 5. 데이터 자동화 (스크래핑 캐싱)
- **IP 차단 방지:** 명시적으로 '길드원 최신화'를 수행할 때만 외부(mgf.gg) 스크래핑 수행
- 10분 서버 캐시를 두어 무분별한 외부 API 호출을 차단하며, 기록 작성 폼에서는 캐싱된 데이터를 DB에서 바로 불러옵니다.

### 6. 판정 및 이미지 공유 (Export)
- 등수 차이에 따라 **양호 / 주의 / 위험** 색상 뱃지 자동 부여
- **공유용 캡처:** 기록 열람 보드에서 캡처 기능 실행 시, 모바일 공유에 최적화된 비율(세로형)로 깔끔한 배경의 이미지가 생성되며 클립보드에 바로 복사하거나 다운로드할 수 있습니다.

## 🗄️ 데이터베이스 스키마 (Supabase)

| 테이블 | 주요 컬럼 | 설명 |
|---|---|---|
| `contents` | id, name, is_server_content | 매왕 길드에서 관리하는 컨텐츠 종류 |
| `records` | id, content_id, title, created_at | 특정 주차의 회차별 결과 |
| `sheet_data` | record_id, power_rank, character_name, content_rank, grade | 멤버별 성적 (`-1` = 미참여) |
| `guild_members` | character_name, power_rank, updated_at | 스크래핑된 길드원 명단 캐시 |
